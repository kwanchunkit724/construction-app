# 13 — Loom 60 秒腳本（錄片用）

一條 60 秒嘅 hook video，send outreach 時附上。**一部 iPhone、一次過錄。**

## 錄之前（30 秒準備）
- iPhone **直度（portrait）**，靜嘅房，開「螢幕錄影」+ 收音。
- App **預先登入 PM `60001001` / `test1234`**，**開咗 DC2026 油塘住宅**（4 個 zone 睇到）。
- 練 2 次先正式錄。錄 3 條，揀最順嗰條。
- 唔使完美，**真實 + 快**就得。

---

## 逐段（秒數 · 螢幕做乜 · 逐句讀）

**0–7s — Hook（唔好掂 app，望鏡頭講）**
> 螢幕：app 首頁 / 項目頁
> 講：「地盤而家啲 daily、PTW、物料，係咪 WhatsApp 加紙簿加 Excel？出 dispute 揾唔返記錄？睇下我哋點解決。」

**7–22s — 進度 + zones（office 即時）**
> 螢幕：撳入項目 → progress tree，慢慢 scroll 幾個 zone 嘅 %
> 講：「判頭、工程師喺電話㩒一下 update 進度，office 即刻見到幾個 zone 嘅情況。唔使再 WhatsApp 問『做到幾多』。」

**22–37s — 物料 + 急件（判頭側痛點）**
> 螢幕：開「物料」→ 指住「急件」嗰項（接駁管 等緊批）
> 講：「判頭叫料，一㩒『急件』，老總部機即刻收到 push。以前 WhatsApp 200 條冚過去就唔見咗,而家係一條有時間戳嘅紀錄。」

**37–48s — Audit trail（大招）**
> 螢幕：撳開一個項目嘅歷史 / 紀錄，指住時間 + 名
> 講：「最值錢係呢度 —— 每一個 action 邊個、幾時、改咗咩,改唔到刪唔到。Dispute 嗰陣打開就係證據,唔使靠記性。」

**48–60s — CTA（望鏡頭收）**
> 螢幕：app icon / 首頁
> 講：「iOS App Store 已經上架,一個月完全免費試。有興趣 WhatsApp 我,我幫你 set 好一個 zone 試下。」

## 錄完點用
1. Loom 免費 tier 上載 → 攞條 share link。
2. 貼入 `10-OUTREACH-DRAFTS` 所有 `[Loom]` 位。
3. Script A / B / C 發 outreach 時，連 link 一齊 send。

## Tips
- 講快啲、語氣自然，唔好讀稿感。
- 手指㩒邊講邊指，等人跟到。
- 60 秒一過就收 —— 短先有人睇完。
- 後備：如果一次過錄唔順，分 5 段錄再 Loom / 剪片砌埋。
