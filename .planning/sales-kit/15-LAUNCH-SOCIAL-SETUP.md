## Step-by-step（我會喺 Chrome 一步步陪你做）

### A. Facebook Page
1. 登入你個人 FB（你嚟做，我唔掂密碼）。
2. 去 **facebook.com/pages/create**。
3. Page name = `CK工程 | Construction App`；Category 揀 `Software` / `Construction Company`。
4. Bio 貼上面「About」。撳 Create。
5. 上傳 **profile pic**（icon-512）+ **cover**（封面圖 screenshot）。
6. Edit Page info → Website = `…/#/sell`、Email、WhatsApp、設定 username `@ckconstructionhk`。
7. 加 CTA button → "Sign Up" → `…/#/sell`。
8. 出 Post 1。

### B. Instagram（建議用手機 app 開，連埋 FB）
1. 手機開 IG → 注冊新帳戶（**唔好用你私人 IG**，開新嘅）。
2. Username `@ckconstructionhk`（同 FB 一樣）。
3. Switch to **Professional / Business account** → category `Software`。
4. Bio 貼上面短 bio + link `…/#/sell`。
5. Profile pic = icon-512。
6. Settings → 連結 Facebook Page（之後一次過出 FB + IG）。
7. 出 Post 1–3。

### C. 連埋（一次過管理）
- FB Business Suite（business.facebook.com）可以一個位排 FB + IG 嘅 post。
- 全部 bio / CTA → `…/#/sell`（有 lead form → /#/mission 潛在客戶）。
