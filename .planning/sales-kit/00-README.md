## Quick Start (TL;DR)

1. **First week**: Read 01-03. Make list of 30 target firms. Cold-DM 10 per day on LinkedIn + WhatsApp.
2. **Land 3 demos**: Use 04 (pitch) + 05 (demo).
3. **Close first 2 paying clients**: Use 06 (pricing) + 07 (objections).
4. **Reinforce**: Build case study, post on socials, ask for referrals.
