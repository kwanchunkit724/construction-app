私隱政策

最後更新：2026 年 5 月 3 日

關春傑工程有限公司（下稱「我們」）開發並營運「建築工程管理」流動應用程式（下稱「本 App」）。 本政策說明我們如何收集、使用、儲存及保護用戶資料。使用本 App 即代表你同意本政策內容。

1. 收集嘅資料

1.1 你直接提供嘅資料

  • 帳號資料：手機號碼（用作帳號識別碼）、姓名、所屬公司、職位／角色、密碼（加密儲存）。
  • 工地操作資料：你建立或更新嘅工地項目、進度、樓層完成情況、備注。
  • 問題報告：問題標題、描述、現場照片、留言。
  • 申請紀錄：你申請加入工地嘅角色、批准／拒絕嘅紀錄、時間戳記。

1.2 自動收集嘅資料

  • 裝置識別碼：用於推送通知（透過 Apple Push Notification service）。
  • 使用紀錄：登入時間、操作時間戳記，用作紀錄追溯。

1.3 我們不收集嘅

  • 位置資料（GPS）
  • 聯絡人
  • 瀏覽紀錄
  • 除問題報告以外嘅相片或檔案
  • 付款／信用卡資料

2. 相機 / 相簿 權限

本 App 喺報告問題時可能需要使用相機（拍攝現場狀況）或讀取相簿（選擇現有相片）。 我們只在你主動報告問題時才會調用相機／相簿，並只上傳你選擇嘅相片。 你可以隨時喺 iOS 設定 → 通知／相機／相片 中關閉相關權限。

3. 推送通知

我們透過 OneSignal 服務發送以下通知：

  • 新問題報告（給對應處理層人員）
  • 問題上呈（給下一層處理人員）
  • 問題已解決（給報告者）
  • 工地申請通過或拒絕（給申請者）
  • 被指派為 PM 或負責人

你可以喺 iOS 設定 → 通知 → 「CK Construction」中關閉推送通知。 我們不會用推送通知做行銷或廣告。

4. 資料儲存與第三方服務

本 App 嘅資料儲存於以下第三方服務嘅基礎設施上：

  • Supabase（資料庫 + 檔案儲存） — 服務器位於美國／歐洲。所有資料以 HTTPS 加密傳輸，密碼以 bcrypt 雜湊儲存。 隱私政策：https://supabase.com/privacy
  • OneSignal（推送通知路由） — 接收用戶 ID 用作識別發送目標，不接收個人資料、相片或工地內容。 隱私政策：https://onesignal.com/privacy_policy
  • Apple（推送通知傳輸 APNs） — Apple 嘅推送服務僅傳遞通知 token，不儲存內容。

5. 資料保留與刪除

  • 帳號資料：保留至帳號被刪除為止。
  • 工地相關資料（進度、問題、相片）：保留作工程紀錄用途，與工地壽命相符。
  • 用戶可隨時聯絡我們要求刪除帳號及相關資料（見第 8 節聯絡方法）。
  • 系統日誌：保留 90 日後自動刪除。

6. 資料分享

我們不會將你嘅資料賣俾或分享俾廣告商或第三方營銷公司。 我們僅在以下情況下分享資料：

  • 同一工地內嘅成員可以見到該工地嘅進度、問題、申請紀錄（基於角色權限）。
  • 系統管理員（Admin）可以見到所有用戶帳號資料（用於管理）。
  • 法律或政府部門合法要求時。

7. 資料保安

  • 所有 API 通訊使用 HTTPS / TLS 加密。
  • 密碼使用 bcrypt 雜湊儲存，絕不明文儲存。
  • 資料庫使用 Row-Level Security（行級安全）防止越權存取。
  • 相片以隨機路徑儲存，需透過已驗證身份方可存取。

8. 兒童隱私

本 App 為工地專業人員設計，不向 13 歲以下兒童提供。我們不會故意收集兒童資料。

9. 你嘅權利

你有權：

  • 查閱我們儲存嘅關於你嘅資料
  • 修改不正確嘅資料
  • 要求刪除你嘅帳號及資料
  • 撤回對推送通知或相機／相簿嘅授權

請聯絡我們（見第 10 節）行使上述權利。

10. 聯絡我們

如有任何關於私隱嘅查詢，請聯絡：

  • 公司名稱：關春傑工程有限公司
  • 電郵：kck980724@gmail.com

11. 政策修訂

我們可能不時修訂本政策。重大變更會喺 App 內或透過電郵通知你。 繼續使用本 App 即表示你接受最新版本嘅政策。

Privacy Policy (English)

Last updated: 3 May 2026

Kwan Chun Kit Construction Ltd. ("we", "us") develops and operates the "Construction Management" mobile application ("the App"). This Privacy Policy explains how we collect, use, store, and protect user data. By using the App, you agree to this policy.

1. Information We Collect

1.1 Information You Provide

  • Account data: phone number (used as account identifier), name, company, role, password (stored encrypted).
  • Project operation data: project items, progress updates, floor completion, notes.
  • Issue reports: issue title, description, on-site photos, comments.
  • Application records: roles applied for, approval/rejection records, timestamps.

1.2 Automatically Collected

  • Device identifier (for push notifications via APNs).
  • Usage timestamps (login, operation times) for audit purposes.

1.3 We Do Not Collect

  • GPS / location data
  • Contacts
  • Browsing history
  • Photos or files outside of issue reporting
  • Payment / credit card data

2. Camera / Photo Library Permissions

The App may request camera or photo library access only when you actively choose to attach photos to an issue report. We only upload photos you explicitly select. You can revoke these permissions at any time in iOS Settings → Notifications / Camera / Photos.

3. Push Notifications

We send push notifications via OneSignal for the following events:

  • New issue reported (to handler-level personnel)
  • Issue escalated (to next-level handler)
  • Issue resolved (to reporter)
  • Project membership approved or rejected (to applicant)
  • You were assigned as PM or task owner

You can disable push notifications at any time in iOS Settings → Notifications → "CK Construction". We never use push notifications for marketing or advertising.

4. Storage and Third-Party Services

App data is stored on the following third-party infrastructure:

  • Supabase (database + file storage) — servers in US / EU. All data transmitted over HTTPS; passwords stored as bcrypt hashes. https://supabase.com/privacy
  • OneSignal (push notification routing) — receives only user identifiers for delivery targeting; does not receive personal data, photos, or project content. https://onesignal.com/privacy_policy
  • Apple (APNs) — only delivers notification tokens; does not store content.

5. Data Retention and Deletion

  • Account data: retained until account deletion.
  • Project data (progress, issues, photos): retained for the duration of the project as construction records.
  • You may contact us at any time to request deletion of your account and associated data.
  • System logs: automatically deleted after 90 days.

6. Data Sharing

We do not sell or share your data with advertisers or marketing companies. We share data only when:

  • Members of the same project can see project progress, issues, and applications (per role-based permissions).
  • System administrators can see all user account data for management purposes.
  • Required by law or government authority.

7. Security

  • All API traffic uses HTTPS / TLS.
  • Passwords are bcrypt-hashed — never stored in plaintext.
  • Database uses Row-Level Security to prevent unauthorized access.
  • Photos stored at randomised paths; access requires authenticated session.

8. Children's Privacy

The App is designed for construction professionals and is not directed at children under 13. We do not knowingly collect data from children.

9. Your Rights

You have the right to:

  • Access the data we hold about you
  • Correct inaccurate data
  • Request deletion of your account and data
  • Withdraw consent for push notifications, camera, or photo library access
